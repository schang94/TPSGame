# TPSGame
플레이어와 적의 코드 재활용

## 개발
- 엔진 : Unity
- 언어 : C#
- 주요 에셋/패키지 : AI Navigation, Cinemachine, Input System

## 조작방법
- 이동 : 마우스 L클릭
- 공격 : 마우스 L클릭

## 특징
- 플레이어와 적은 오브젝트 풀링을 이용
- 플레이어와 적은 같은 스크립트를 사용하여 코드 재활용
- 적은 PathPoint를 따라 이동
